# AI Study Assistant

AI Study Assistant-ը կրթական վեբ ծրագիր է, որը օգնում է աշակերտներին
հասկանալ դասերը՝ տալով պարզ բացատրություններ և օրինակներ։

## Հնարավորություններ
- Հարցերի բացատրություն քայլ առ քայլ
- Դպրոցական պարզ լեզու
- Կրթական նպատակով օգտագործում

## Ովքերի համար է
- Աշակերտներ
- Ուսուցիչներ
- Դպրոցներ

## Տեխնոլոգիաներ
- HTML
- CSS
- JavaScript

## Նպատակ
Օգնել սովորել, ոչ թե արտագրել։
